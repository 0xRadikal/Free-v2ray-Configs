## 📖 How to Use

### 🤖 Android — v2rayNG / NekoBox / Hiddify

1. Open **v2rayNG** → tap **≡** → **Subscription Group Settings**
2. Tap **+** → paste the subscription link → Save
3. Pull down to refresh → tap **▶** to connect

### 💻 Windows — v2rayN / NekoRay / Hiddify

1. Open **v2rayN** → **Servers** menu → **Add subscription**
2. Paste the subscription link → OK
3. Right-click tray icon → **Update all subscriptions**
4. Select a server → press **Enter** to connect

### 🍎 iOS — Shadowrocket / FoXray / Hiddify

1. Open **Shadowrocket** → tap **+** (top right)
2. Select **Subscribe** → paste the subscription link → Done
3. Tap the config → Connect

### 🐧 Linux — NekoRay / Clash Meta / Hiddify

1. Open **NekoRay** → **Program** → **Add profile from URL**
2. Paste the subscription link → OK
3. Right-click a profile → **Start**
