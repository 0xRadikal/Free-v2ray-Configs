## 🌐 این ریپو چیه؟

این ریپوزیتوری **کانفیگ رایگان V2Ray** و **لینک ساب‌اسکریپشن** ارائه می‌ده که هر **۳۰ دقیقه** به صورت کاملاً خودکار از طریق GitHub Actions آپدیت می‌شن. تمام کانفیگ‌ها جمع‌آوری، تست و بر اساس پروتکل دسته‌بندی شدن — بدون ثبت‌نام، بدون پرداخت، بدون تبلیغ.

## 📖 آموزش استفاده

### 🤖 اندروید — v2rayNG / NekoBox / Hiddify

1. **v2rayNG** رو باز کن ← روی **≡** بزن ← **Subscription Group Settings**
2. روی **+** بزن ← لینک ساب‌اسکریپشن رو پیست کن ← ذخیره
3. صفحه رو به پایین بکش تا آپدیت بشه ← روی **▶** بزن تا وصل بشی

### 💻 ویندوز — v2rayN / NekoRay / Hiddify

1. **v2rayN** رو باز کن ← منوی **Servers** ← **Add subscription**
2. لینک ساب‌اسکریپشن رو پیست کن ← OK
3. روی آیکون tray راست‌کلیک کن ← **Update all subscriptions**
4. یه سرور انتخاب کن ← **Enter** بزن تا وصل بشی

### 🍎 آیفون / iOS — Shadowrocket / FoXray / Hiddify

1. **Shadowrocket** رو باز کن ← روی **+** (بالا راست) بزن
2. **Subscribe** رو انتخاب کن ← لینک ساب‌اسکریپشن رو پیست کن ← Done
3. روی کانفیگ بزن ← Connect

### 🐧 لینوکس — NekoRay / Clash Meta / Hiddify

1. **NekoRay** رو باز کن ← **Program** ← **Add profile from URL**
2. لینک ساب‌اسکریپشن رو پیست کن ← OK
3. روی یه پروفایل راست‌کلیک کن ← **Start**

## 🛡️ پروتکل‌های پشتیبانی‌شده

تمام کانفیگ‌های این ریپو از پروتکل‌های زیر پشتیبانی می‌کنن و با هسته‌های **XRay-core** و **Sing-box** سازگار هستن:

| پروتکل | توضیح |
|--------|-------|
| **VLESS** | پروتکل سبک و پرسرعت با پشتیبانی از XTLS و Reality |
| **VMess** | پروتکل اصلی V2Ray با پشتیبانی گسترده |
| **Trojan** | ترافیک رو به شکل HTTPS نشون می‌ده — سخت‌ترین برای تشخیص |
| **Shadowsocks (SS)** | پروتکل پروکسی رمزنگاری‌شده سریع و ساده |
| **Hysteria2** | پروتکل UDP برای شبکه‌های با تأخیر بالا بهینه شده |
| **TUIC** | پروتکل مبتنی بر QUIC با تأخیر کم و پهنای باند بالا |
| **Reality** | پسوند VLESS که TLS واقعی رو شبیه‌سازی می‌کنه — غیرقابل تشخیص |
| **WireGuard** | پروتکل VPN مدرن، فوق‌العاده سریع و امن |
